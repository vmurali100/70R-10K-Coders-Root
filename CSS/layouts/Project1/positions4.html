As we can see here the second div is positioned as fixed with an offset properties as left and right set to zero. That means it is removed from its normal position and is placed at the top right of the view port as a fixed positoned elemnt alwasys takes the view port of the browser window as a reference.

Here as we scroll the page we can see that the second div is fixed at the top right of the page and it doesn't scroll along with the page.
