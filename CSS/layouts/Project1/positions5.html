Sticky positioned element behaves some what same as a relative element but when the element reaches its offset value provided it behaves as a fixed element.So bascially sticky element is a combination of both relative and a fixed element.

In this page you can see we have positioned #one sticky and given an offest valur of top:0 , which means that the elemnt behaves as a relative.,that means it is in exactly at same position as a normal element behaves but as the elemment reaches its offset position it is fixed at that position only and is not scrolled with the page.
