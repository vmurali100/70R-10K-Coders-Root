Here We can see that the absolutely positioned element #one is removed from the normal flow and is positioned at the offset given making the view port as a reference.

So the conclusion is whenever an element is absolutely positioned it is taken away from its normal position and everything renders as if the absolutely positioned element is not at all there.

Generally when an element is positioned absolutely , it takes the reference of the view port or the browser window. Here we have given its parent the position:relative , so here it took the reference of its parent and aligned according to that given offset from its parent.
